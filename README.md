Для питания индикаторных ламп используется напряжение ~180В, поэтому нельзя касаться при работе устройства элементов С1, D2, R8, R7, U2, U3, U4.<br/>
Фьюзы ATtiny44: low: `0xE2`, high: `0xDF`, extended: `0xFF`
